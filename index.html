What is HTML?

HTML (Hypertext Markup Language), is the languange the browsers can understand and show us somethng we won't. This text for example, was wrote with this languange, where you can just type something or put images, buttons, and personalize them.

How does it work?

HTML works with Tags and Elements. For example.
I can make a word becomes Italic, by using a tag the letter i inside < and >. the element inside these Symbols define the tag, and these tags make the page look like what we want. If i want to jump line in HTML, i cannot just press enter, i need to put a tag that allows the text be be shown in next line.

And everything follows the same concepts.
